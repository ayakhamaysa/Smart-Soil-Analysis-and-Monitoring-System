# Hardware

This folder contains the hardware components and wiring used in the Smart Soil Analysis and Monitoring System.

## Components

- ESP32 Development Board
- DHT22 Temperature and Humidity Sensor
- DS18B20 Soil Temperature Sensor
- HW-080 Soil Moisture Sensor
- GPS NEO-6M Module
- Smart RC Vehicle
